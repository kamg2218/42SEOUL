printf 함수

% [flags][width][.precision][size] type

1. flags
    - #   : 플래그가 o, x, #X 와 함께 사용되는 경우, 0이 아닌 출력 값에 접두사 0x, 0X를 추가한다.
    - -   : 왼쪽에 맞춰 표시
    - +   : 양수에 '+' 표시
    - 0   : 공백을 0으로 표시한다. (최소 너비에 도달할 때까지 0 추가)
            -, 0 둘 다 표시되는 경우, 0 무시한다.
            정수에 대하여 0이 있고, width도 지정되어 있는 경우 0 무시한다.
            부동 소수점 형식에 대해 지정된 경우, 0x 또는 0x 접두사 뒤의 가수 앞에 붙는다.
    - ' ' : 부호가 있는 양수 출력 값에 공백을 접두사로 추가한다. (부호 자리)
            만약 '+' flag가 있는 경우 공백은 무시된다.

2. width
    - 최소 출력 너비 값
        - 값이 작은 경우, 츌력 값이 잘리지 않는다.
        - 값이 큰 경우, 필드를 확장한다.

3. .precision
    - 전체 자릿수 사양
        - 0 : 변환하려는 값이 0이면 문자는 출력되지 않는다.
        - * : int 인수가 값을 제공해야 하고, 형식이 지정되는 값 앞에 전달인자로 입력되어야 한다.
              만약, 정수 값이 음수인 경우 무시한다.
        - 만약 어떤 인자도 사용되지 않는 경우, 0으로 간주된다.
        type 에 따른 적용
        - c : 전체 자릿수가 적용되지 않는다.
        - s : 전체 자릿수가 출력할 최대 문자 수를 지정한다. 초과하는 문자는 출력되지 않는다.
        - d, i, u, o, x, X : 최소 자릿수를 지정한다. 부족한 출력 값에는 0을 출력하고, 자릿수가 초과하는 경우 그대로 출력한다.
        - f : 전체 자릿수 값이 소수점 뒤의 자릿수를 지정하고, 반올림한 값이 마지막 자릿수의 값이 된다.

4. size
    - 인수 길이 한정자
    - h     : short int/short unsigned int
    - hh    : char/unsigned char
    - l     : long double / long int/long unsigned int
    - ll    : long long int/unsigned long long int

5. type
    - 형식 변환 지정자
    - c : char / 단일 바이트 문자
    - C : char / 와이드 문자 (와이드 문자의 크기는 항상 고정되어 있어 국제 공용 문자 집합으로 단순하게 프로그래밍 할 수 있다.)
    - s : string / 멀티 바이트 문자
    - S : string / 와이드 문자열
    - d : 부호 있는 10진수 정수
    - i : 부호 있는 10진수 정수
    - u : 부호 없는 10진수 정수
    - o : 부호 없는 8진수 정수
    - x : 부호 없는 16진수 정수이고, "0123456789abcdef" 사용한다.
    - X : 부호 없는 16진수 정수이고, "0123456789ABCDEF" 사용한다.
    - e : 부호 있는 지수 형태 출력, [-]d.ddde[+,-]ddd
    - E : 부호 있는 지수 형태 출력, [-]d.dddE[+,-]ddd
    - f : 부호 있는 10진수 실수, [-]ddd.dddd
    - F : 부호 있는 10진수 실수, [-]ddd.dddd
    - p : 16진수로 된 주소

6. 반환값
    - 출력한 문자의 수 
        또는 출력 에러나 다른 에러가 발생한 경우 음수를 출력한다.
    - 널 문자를 포함하지 않는 출력한 문자의 개수
